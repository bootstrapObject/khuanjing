### 快环境项目说明

    1.页面大概13个左右分别为 登录页、文件管理、数据库、监控、CPU监控、内存监控、磁盘io监控、网络监控、一键放置网站、回收站、退出

      jquery 3+
      adminlte.js
      bootstrap-table.js  1.13.5
      echarts.js 3+
      版本管理 svn
